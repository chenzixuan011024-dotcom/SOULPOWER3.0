Build meaningful matches based on values and interests

SoulPower helps people connect through shared interests, personal values, and communication style. This page is a lightweight prototype intended for classroom demonstration and sprint review evidence.

How to Get Started

1) Sign up and complete a short interest survey. 2) Browse the user catalogue and shortlist profiles. 3) Try one intelligent match. 4) If you both accept, move to the live chat.

Note: This is a front‑end only demo (no database or authentication backend).

About Us

SoulPower is a concept dating platform that prioritises compatibility signals—interests, values, and communication style—over appearance alone. This prototype illustrates the intended user experience.

Implementation notes for a production build: connect authentication (OAuth/JWT), persist profiles and chats in a database, and enable real‑time messaging via WebSocket/RTC with safety moderation and reporting.
